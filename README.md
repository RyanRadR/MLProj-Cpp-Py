# mlproj-cpp-py

mlproj-cpp-py is a C++ and Python project intended to try large, modular, multi-file development using AI assistance.

## Goals
- Build a codebase with modularity
- Use C++ for performance critical components
- Use Python for scripting, tooling, testing, and ML portions
- Upload to GitHub for documentation

## Initial Structure
- 'cpp/' - C++ source and headers
- 'python/' - Python package and entry points
- 'tests/' - test suite
- 'docs/' - design notes and architecture docs
- 'scripts/' - helper scripts
